# Analog Clock with HTML, CSS and JavaScript

A simple analog clock in the browser with moving hands for hours, minutes, and seconds.

## Project Structure

- `index.html` – HTML layout of the clock
- `style.css` – Styling for the clock face and hands
- `analogClock.js` – JavaScript to move the hands

## Features

- Second hand moves every second
- Minute and hour hands update smoothly
- Clock starts automatically when the page loads

## Usage

1. Save all files in one folder
2. Open `index.html` in your browser
3. The clock runs immediately

## Code Explanation

- Gets the current time using `new Date()`
- Calculates rotation angles for each hand
- Rotates hands using `transform: rotate(...)`
- Updates every second with `setInterval()`

## Requirements

- Modern browser
- No additional libraries needed
